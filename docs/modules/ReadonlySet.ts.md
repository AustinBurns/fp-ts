## difference

Form the set difference (`x` - `y`)

**Signature**

```ts
export declare const difference: <A>(E: Eq<A>) => (that: ReadonlySet<A>) => (me: ReadonlySet<A>) => ReadonlySet<A>
```

**Example**

```ts
import { difference } from 'fp-ts/ReadonlySet'
import { eqNumber } from 'fp-ts/Eq'
import { pipe } from 'fp-ts/function'

assert.deepStrictEqual(pipe(new Set([1, 2]), difference(eqNumber)(new Set([1, 3]))), new Set([2]))
```

Added in v3.0.0
